אתה מפתח אפליקציות ראשי עם 20 שנות ניסיון, 
מבקשים ממך לבנות אפליקציית אזורי זמן עולמיים. 


מטרה:
לבנות אפליקציית React המציגה שעות מקומיות של ערים פופולריות בעולם.
API:
time.now Developer API
https://time.now/developer
________________________________________
אפליקציה עם שלושה עמודים. 
1. עמוד ראשי (בית)
●	הצגת גריד של ערים פופולריות (10 ערים לפחות)
לדוגמה: Tel Aviv, London, New York, Tokyo, Paris
●	לכל עיר להציג:
○	שם העיר
○	שעה מקומית נוכחית
○	אזור זמן (timezone)
●	לחיצה על עיר גורמת לעבור לעמוד פרטי עיר
________________________________________
2. עמוד פרטי עיר (פרטי העיר)
●	הצגת מידע מפורט:
○	שעה מקומית
○	תאריך
○	אזור זמן מלא (IANA)
●	כפתור חזרה לעמוד הראשי
________________________________________
3. עמוד )עלינו)
●	עמוד סטטי הכולל:
○	הסבר קצר על האפליקציה
○	ציון מקור הנתונים (time.now API)
________________________________________
דרישות טכניות
●	React עם Vite typescript
●	React Router (Home / City / About)
●	שימוש ב־fetch כמובן עם asyc/await, try-catch
●	useState + useEffect
●	עיצוב בסיסי (CSS Grid / Flex) ללא bootstrap/tailwindcss
●	טיפול ב־loading ו־error
________________________________________
מגבלות
●	אין להשתמש בספריות זמן חיצוניות (moment, dayjs וכו׳)
●	אין שימוש ב־UI frameworks (MUI, AntD, Bootstrap)
________________________________________
בנוסף, לעשות: 
●	רענון שעה כל 60 שניות
●	אינדיקציה יום / לילה לפי השעה - אפשר לעשות עם emoji 
●	כפתור רענון ידני (רק בעמוד cityDetails)
●	darkmode/lightmode 
________________________________________
קריטריונים לבדיקה
●	קוד נקי וקריא
●	חלוקה נכונה לקומפוננטות
●	שימוש נכון ב־React Router
●	טיפול נכון בתקלות API
●	התאמה מלאה לדרישות
●	ריספונסיביות מלאה
